보핸님, 기획하신 지적장애인 생애주기별 지원 플랫폼(Soom-Garden, 숨가든)의 핵심 기능인 **생애주기별 맞춤형 입력 폼**을 실제 웹 애플리케이션으로 구현해 보실 수 있도록 대화형 UI 프로토타입으로 설계했습니다.  
지적장애인 당사자 및 보호자, 특수교사, 사회복지사 등 다양한 이해관계자의 특성을 고려하여 다음과 같은 혁신적인 UX 요소를 반영했습니다.

### **주요 설계 특징**

1. **쉬운 말(Easy-Read) 모드 토글:** 지적장애인 당사자나 긴장한 보호자가 쉽게 이해할 수 있도록 전문 용어를 일상적이고 친숙한 질문과 이모지로 실시간 전환합니다.  
2. **이해관계자별 입력 최적화:** 폼 상단에서 현재 작성 주체(당사자, 가족, 교사/치료사)를 선택하여 그에 최적화된 문항 스타일을 제공합니다.  
3. **AI 실시간 도움말 가이드:** 입력란마다 무엇을 써야 할지 막막할 때 AI가 구체적인 작성 예시를 제공하여 기록의 질을 높입니다.  
4. **전 생애주기 및 행동지원(PBS) 통합 탭:** 영유아기부터 노년기, 그리고 상시 필요한 행동지원 기록까지 한 화면에서 부드럽게 전환하며 살펴볼 수 있습니다.

### **UI 설계의 상세 컴포넌트 해설 (DB 설계 연계용)**

위 HTML 프로토타입에 내재된 **입력 폼 컴포넌트별 세부 속성 및 UX 설계 가치**를 정리해 드립니다.

#### **1\. 영유아기 (0\~6세) 입력 폼 설계**

* **스크리닝 라디오 버튼 (infant-motor):** 3점 척도를 활용해 K-DST(한국 영유아 발달선별검사)의 간이식 분석 체계를 채택했습니다.  
* **치료사 연계 주관식 텍스트 필드:** 아동의 언어치료와 감통치료 과정에서 관찰되는 비정형 데이터를 자유롭게 입력하면, 우측의 **AI 라이브 코치**가 인성 및 대칭 발달 데이터셋으로 구조화하여 정기 리포트에 즉각 반영하도록 설계되었습니다.

#### **2\. 학령기 (7\~18세) 입력 폼 설계 (Default Active)**

* **IEP 슬라이더 (iepRange):** 학기 초 특수학급 교사와 조율한 목표치(예: "천 원 지폐 3장을 세어 지불하기")에 대한 주관적 달성 여부를 슬라이더 조작을 통해 한눈에 정량 수치로 수집합니다.  
* **통합 학급 어울림 텍스트 필드:** 학교 친구들과 마주하는 사회성 이벤트, 자폐 스펙트럼 성향이나 충동 조절 징후를 부모와 특수교사가 공동으로 작성하여 알림장을 대신하도록 만들었습니다.

#### **3\. 성인기 (19\~39세) 입력 폼 설계**

* **근태 상태 가독 카드:** 직무 훈련 근태 현황을 한눈에 식별할 수 있도록 아이콘화된 카드를 두어 접근성을 높였습니다.  
* **자립/가사 자립 수준 서술형:** 스마트폰 길찾기 사용, 편의점 카드 결제 등 독립 자립 활동에 특화된 서식을 제공합니다.

#### **4\. 중장년/노년기 (40세 이상) 입력 폼 설계**

* **조기 노화 및 치매 징후 체크리스트:** 지적장애인에게 흔히 나타나는 급격한 조기 퇴행성 질환 및 치매 징후(신변 뒤처리 실수 등)를 쉽게 기록할 수 있도록 선택형 체크박스 인터페이스를 구축했습니다.  
* **사후 신탁 준비 드롭다운:** 돌봄 공백의 큰 요인인 '부모 사후 자녀 돌봄' 문제를 플랫폼 수준에서 실시간 케어 매니징할 수 있도록 사전 설계 진행 여부를 확인합니다.

#### **5\. 도전행동(PBS) 일지 설계**

* **행동 기능 분석용 ABC 입력 설계:** 긍정적 행동지원을 완벽히 수행하기 위해 유형(Behavior) \-\> 원인 및 배경(Antecedent) \-\> 사후 대응(Consequence)의 과학적 인과 분석 필드를 구성해 플랫폼 AI가 행동 소강 패턴을 스스로 머신러닝할 수 있게 도왔습니다.
