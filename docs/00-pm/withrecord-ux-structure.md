## A.1 Hero 섹션 (핵심 가치 메시지)

### 목적
첫 5초 안에 "이것이 무엇인가 / 누구를 위한 것인가 / 왜 지금 필요한가"를 전달.

### 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   기록은 끊기지 않고,                                    │
│   권한은 당사자에게 자란다                               │
│                                                          │
│   지적장애인 가족을 위한 평생 기록 +                     │
│   자기결정권 플랫폼                                      │
│                                                          │
│   [무료로 시작하기]  [기관 도입 상담]                    │
│   "신규 가입 1시간 안에 첫 인계서 PDF를 받아보세요"      │
│                                                          │
│   ───────────────────────────────────                    │
│   ✓ 한국지적발달장애인복지협회 추천 (예정)               │
│   ✓ ISMS-P 인증 (예정) | ✓ AAC 픽토그램 지원            │
│                                                          │
│   [오른쪽: 일러스트 — 다세대 가족 +                       │
│           휴대폰 화면(타임라인 + AAC) 목업]              │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 주요 콘텐츠
- 헤드라인 (h1): "기록은 끊기지 않고, 권한은 당사자에게 자란다"
- 서브 헤드라인: "지적장애인 가족을 위한 평생 기록 + 자기결정권 플랫폼"
- Primary CTA: [무료로 시작하기] — 보호자 가입 플로우
- Secondary CTA: [기관 도입 상담] — 시설장/B2B 문의 폼
- 보조 메시지: "신규 가입 1시간 안에 첫 인계서 PDF를 받아보세요" (First Value 약속)
- 신뢰 배지: 협회 추천, ISMS-P 인증, AAC 지원

### UI 컴포넌트
- 풀스크린 좌우 분할 (좌: 텍스트 / 우: 일러스트)
- 모바일에서는 일러스트 위 + 텍스트 아래
- Primary CTA 버튼: 강조색 (브랜드 메인 컬러 `#3B82F6`), 큰 사이즈 (높이 56px)
- Secondary CTA 버튼: 보더만 있는 보조 스타일

### 메시지 톤
- **감성 + 신뢰**: 시적 헤드라인 + 즉시 행동 가능한 CTA
- 전문 용어 회피, "가족" "평생" "자라다" 등 따뜻한 어휘

## A.2 Problem 섹션 (3가지 페인포인트)

### 목적
방문자가 "이건 내 이야기다"라고 공감하게 만드는 자기 인식 트리거.

### 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   왜 지금 위드레코드인가요?                              │
│                                                          │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐              │
│   │  📁      │  │  ✍️       │  │  🤝      │              │
│   │ 기록     │  │ 행정     │  │ 자기     │              │
│   │ 단절     │  │ 부담     │  │ 결정권   │              │
│   │          │  │          │  │ 배제     │              │
│   │ 기관 이동│  │ 일 30분  │  │ 평생     │              │
│   │ 마다     │  │ 일지     │  │ 보호자   │              │

│   │          │  │          │  │          │              │
│   │"30년 모은│  │"퇴근 후도│  │"엄마, 이건│              │
│   │ 노트를   │  │ 일하는   │  │ 내가     │              │
│   │ 누가     │  │ 게 진짜  │  │ 할게요"  │              │
│   │ 다 봐줘?"│  │ 힘들어요"│  │          │              │
│   └──────────┘  └──────────┘  └──────────┘              │
│                                                          │
```

### 주요 콘텐츠

| 카드 | 제목 | 수치 | 페르소나 인용구 |
|------|------|------|---------------|
| 1 | **기록 단절** | "기관 이동마다 2주씩 적응 기간" | "30년 모은 노트를 누가 다 봐주겠어요?" — 박순영 (보호자) |
| 2 | **행정 부담** | "일 평균 30분 일지에 묶인 일상" | "퇴근하고 30분 더 일하는 게 진짜 힘들어요." — 이수진 (활동지원사) |
| 3 | **자기결정권 배제** | "평생 보호자 동의에 의존" | "엄마, 이건 내가 할게요." — 김민지 (당사자) |

### UI 컴포넌트
- 3열 카드 그리드 (모바일 1열)
- 각 카드: 큰 픽토그램 + 굵은 제목 + 수치 한 줄 + 페르소나 인용
- 호버 시 카드 살짝 들림 + 그림자

### 메시지 톤
- **공감 우선**: 데이터보다 페르소나 목소리를 먼저
- **솔직한 수치**: 30분, 2주, 평생 — 구체성으로 신뢰

## A.3 Solution 섹션 (주요 기능 4개)

### 목적
"이 도구가 어떻게 내 문제를 해결하나"를 4개 핵심 기능으로 입증.

### 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│   위드레코드는 이렇게 해결합니다                         │
│                                                          │
│   ┌─────────────────────────────────────────────┐       │
│   │  [좌: 스크린샷 목업]   1️⃣ 통합 타임라인     │       │
│   │                       영유아부터 성인까지   │       │
│   │                       단 하나의 기록선     │       │
│   │                       [자세히 보기 →]      │       │
│   └─────────────────────────────────────────────┘       │
│   ┌─────────────────────────────────────────────┐       │
│   │  2️⃣ AI 음성 일지       [우: 스크린샷 목업]  │       │
│   │  말 한 마디로 30초                          │       │
│   │  → 자동 카테고리                            │       │
│   │  [자세히 보기 →]                            │       │
│   └─────────────────────────────────────────────┘       │
│   ┌─────────────────────────────────────────────┐       │
│   │  [좌: 스크린샷 목업]   3️⃣ 3분 인계서        │       │
│   │                       기관 전환의 2주를    │       │
│   │                       3분으로 줄이다       │       │
│   └─────────────────────────────────────────────┘       │
│   ┌─────────────────────────────────────────────┐       │
│   │  4️⃣ AAC 동의 UI        [우: 스크린샷 목업]  │       │
│   │  당사자가 직접 그림으로                     │       │
│   │  Yes/No                                     │       │
│   └─────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────┘
```

### 주요 콘텐츠

| # | 기능명 | 핵심 메시지 | 보조 설명 |
|---|--------|------------|----------|
| 1 | **통합 타임라인** | "영유아부터 성인까지, 단 하나의 기록선" | 생애주기 5단계 색상 구분, 카톡 사진 1만장 자동 정리 |
| 2 | **AI 음성 일지** | "말 한 마디로 30초, 자동 카테고리" | 한국어 비표준 발화 학습, 식사·투약·행동·정서·학습 5종 자동 분류 |
| 3 | **3분 마스터 인계서** | "기관 전환의 2주를 3분으로" | PDF/시스템 동시 전달, 권한 이양 워크플로 자동화 |
| 4 | **AAC 동의 UI** | "당사자가 직접 그림으로 Yes/No" | 픽토그램+음성안내+2단계 확인, 권한 이행 단계 시뮬레이션 |

### UI 컴포넌트
- 좌우 교차 배치 (Zigzag layout)
- 각 섹션: 스크린샷 목업 (1/2 폭) + 텍스트 (1/2 폭)
- "자세히 보기" 링크 → 기능별 상세 페이지 (Phase 2)

### 메시지 톤
- **숫자 + 구체적 행동**: "30초", "3분", "5종", "단 하나"
- 추상어 회피, 실제 사용자 행동 동사 사용 ("말한다", "분류한다", "전달한다")

## A.4 Persona Benefit 섹션 (이해관계자별 혜택)

### 목적
다양한 이해관계자가 "내 입장에서 무엇이 좋은가"를 즉시 확인.

### 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│   누구에게 좋은가요?                                     │
│                                                          │
│   [당사자] [보호자] [활동지원사] [사회복지사] [시설장]  │ ← 탭
│   ─────────                                              │
│                                                          │
│   👨‍👩‍👧 보호자 (선택된 탭)                                  │
│                                                          │
│   ✓ 카톡 사진 1만 장을 14일 만에 자동 시기 정리          │
│   ✓ 자녀 18세 권한 이양 단계별 가이드 + 변호사 무료 상담 │
│   ✓ 신규 기관에 인계서 PDF 1-Click 전달                  │
│   ✓ 다른 활동지원사 권한 만료 D-7 자동 알림              │
│   ✓ 자녀 사망/탈퇴 시 30일 내 완전 삭제 보장             │
│                                                          │
│   "30페이지짜리 인계서를 다시 쓰지 않아도 돼요"          │
│                                       — 박순영 (54세)    │
│                                                          │
│   [무료로 시작하기 →]                                    │
└─────────────────────────────────────────────────────────┘
```

### 주요 콘텐츠 (5개 탭별)

**탭 1. 당사자 (쉬운말로 표시)**
- ✓ 그림으로 답할 수 있어요
- ✓ 어른이 되면 나도 정할 수 있어요
- ✓ 내가 누구한테 일기를 보여줄지 정해요

**탭 2. 보호자**
- ✓ 카톡 사진 1만 장을 14일 만에 자동 정리
- ✓ 자녀 18세 권한 이양 단계별 가이드
- ✓ 신규 기관 인계서 1-Click 전달
- ✓ 권한 만료 D-7 자동 알림
- ✓ 30일 내 완전 삭제 보장

**탭 3. 활동지원사**
- ✓ 일지 작성 시간 30분 → 2분
- ✓ 신규 이용자 3분 인계서로 첫날부터 안전한 케어
- ✓ 본인 부담금 0원 (기관/보호자 결제)
- ✓ 계약 종료 시 데이터 자동 회수 → 책임 부담 없음
- ✓ 오프라인 산간 캠프에서도 음성 기록 가능

**탭 4. 사회복지사**
- ✓ 케이스의 교육/의료/돌봄/자립 4영역 한눈에
- ✓ 권한 범위 안의 데이터만 안전 조회
- ✓ 사례관리 회의 자료 자동 생성
- ✓ 행복e음과의 데이터 보완 사용

**탭 5. 시설장**
- ✓ 이용자 12~80명 통합 관리 대시보드
- ✓ 활동지원사 권한 일괄 갱신/회수 (영향 범위 미리보기)
- ✓ 분기 인계서 정산 + B2G 바우처 자동 청구
- ✓ ISMS-P 인증으로 시설 신뢰도 강화

### UI 컴포넌트
- 탭 컴포넌트 (5개, 기본 선택: 보호자)
- 탭 콘텐츠: 체크 리스트 5~7개 + 페르소나 인용 + Tab별 CTA

### 메시지 톤
- **You-Centered**: "당신은~" 화법, 혜택 문장은 동사로 시작
- 당사자 탭만 쉬운말로 작성
