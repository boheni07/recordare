## 공통 설계 원칙

```
1. 단순성 (Simplicity)     16px 아이콘에서도 식별 가능한 심플한 형태
2. 확장성 (Scalability)    앱 아이콘 → 현수막까지 품질 손실 없이 적용
3. 접근성 (Accessibility)  색각 이상자도 형태만으로 인식 가능한 구조
4. 온기 (Warmth)           딱딱하지 않은 곡선 위주, 사람 중심 감성
5. 생애주기 은유           성장·이어짐·흐름 중 하나를 형태에 내포
```

## Direction A — 잎맥 타임라인 (Leaf Vein Timeline)

### 컨셉

나뭇잎 안의 잎맥 구조로 생애주기 타임라인을 표현.
중심 줄기(당사자의 삶)에서 가지(기관·기록·이해관계자)가 뻗어나오는 형태.
잎 전체는 성장과 자립의 완성을 상징.

### 스케치

```
                    ·
                   /|\
                  / | \
                 /  |  \
                /   |   \
               ·    |    ·
              /|    |    |\
             / |    |    | \
            ·  ·    |    ·  ·
                \   |   /
                 \  |  /
                  \ | /
                   \|/
                    ·

         ╔═══════════════════╗
         ║   Recordare        ║
         ║   레코다레          ║
         ╚═══════════════════╝
```

**가로형 조합**
```
  ┌─────┐
  │ ╱│╲ │   Recordare
  │╱ │ ╲│   레코다레
  └─────┘
   [잎맥]    [워드마크]
```

**앱 아이콘형 (1:1)**
```
  ╭───────╮
  │  ·/·  │
  │ /│╲   │
  │╱ │ ╲  │
  │  ·    │
  ╰───────╯
```

### 컬러 적용

| 요소 | 컬러 | 비고 |
|------|------|------|
| 잎 외곽선 | 딥 그린 `#2D6A4F` | Primary |
| 중심 줄기 | 딥 그린 `#2D6A4F` | Primary |
| 측면 가지 | 소프트 코랄 `#E07A5F` | Accent — 당사자 강조 |
| 워드마크 | 차콜 `#2D2D2D` | 기본 / 흰 배경 위 |

**색각 이상 대응**: 잎맥 굵기 차이로 중심-가지 구분 (색 없이도 인식)

### 변형 버전

```
[단색 버전]         [역전 버전]          [축약 아이콘]
  딥 그린 단색        흰 배경 → 딥 그린     R 안에 잎맥
  모노크롬 가능        배경 → 잎맥 흰색      16px 대응용
```

### 강점 / 약점

| 강점 | 약점 |
|------|------|
| 생애주기 은유가 직관적 | 소형(16px)에서 잎맥 디테일 손실 가능 |
| 자연·성장·따뜻함 동시 표현 | 환경 브랜드로 오해 가능성 |
| 컬러 포인트 적용 유연 | 경쟁 유사 로고 사전 조사 필요 |
| 로고타입 없이도 독립 사용 가능 | |

## Direction B — 연결 고리 성장 (Growing Links)

### 컨셉

크기가 점점 커지는 원(○) 4개가 선으로 이어진 형태.
좌→우로 갈수록 원이 커지며 성장을 표현.
각 원은 생애주기 단계(영유아→학령기→전환기→성인기)를 상징.
원들을 잇는 선은 기록의 연속성을 표현.

### 스케치

```
  ○──○──○──○
  ↑  ↑  ↑  ↑
  영  학  전  성
  유  령  환  인
  아  기  기  기

  (원의 크기가 좌→우로 점점 커짐)
```

**비율 스케치**
```
  ·─·──·───·────·
  ↑
  (각 원: 지름 8 : 12 : 18 : 26 비율)
```

**가로형 조합**
```
  · ─ · ─ · ─ ·    Recordare
                    레코다레
```

**앱 아이콘형 (1:1)**
```
  ╭───────────╮
  │  ·─·─·─● │
  │          │
  ╰───────────╯
  (마지막 원을 가장 크게 강조)
```

### 컬러 적용

| 요소 | 컬러 | 비고 |
|------|------|------|
| 첫 원 (영유아기) | 크림 `#F5F0E8` + 그린 테두리 | 시작·보호 |
| 중간 원 2개 | 그린 그라데이션 | 성장 |
| 마지막 원 (성인기) | 딥 그린 `#2D6A4F` 채움 | 자립 완성 |
| 연결선 | 딥 그린 `#2D6A4F` | |

### 강점 / 약점

| 강점 | 약점 |
|------|------|
| 성장·연속성 직관적 | 단순해서 기억에 오래 남지 않을 수 있음 |
| 소형에서도 형태 유지 | 생애주기 4단계 설명이 필요할 수 있음 |
| 기술 플랫폼 감성 표현 | 일반 IT 서비스 로고와 유사 가능성 |
| 색각 이상에도 형태로 인식 | |

## Direction C — R 이니셜 + 심장 (Cor)

### 컨셉

Recordare 어원 "re + cor(심장·마음)"에서 착안.
이니셜 'R' 자 내부에 하트(♥) 또는 심장 형태를 내포.
"마음에 새기다"는 의미를 시각적으로 표현.
클래식하고 품격 있는 느낌으로 B2G·사회적기업 포지셔닝에 유리.

### 스케치

**버전 C-1: R 내부 하트**
```
  ┌───────────────┐
  │  ██╗          │
  │  ██╔══██╗     │
  │  █████╔╝♥     │
  │  ██╔═██╗      │
  │  ██║  ██╗     │
  │  ╚═╝  ╚═╝     │
  └───────────────┘
  R의 오른쪽 곡선 부분에 하트 형태 내포
```

**버전 C-2: 심장 박동선 + R**
```
  ╭──────────────────╮
  │  R ∿∿♥∿∿         │
  │  (EKG 심전도 라인) │
  ╰──────────────────╯
  R 옆에 심장 박동선(EKG) 연결 — "살아있는 기억"
```

**버전 C-3: R 자 하단 곡선 변형**
```
      ██╗
      ██╔══╗
      █████╣    ← 상단 곡선을 하트 형태로
      ██╔═██╗
      ██║  ╚♡   ← 하단 꼬리를 하트로 마무리
      ╚═╝
```

**가로형 조합**
```
  [R♥]   Recordare
          레코다레
```

**앱 아이콘형 (1:1)**
```
  ╭───────╮
  │       │
  │  R♥   │
  │       │
  ╰───────╯
  딥 그린 배경 + 크림 R♥
```

### 컬러 적용

| 요소 | 컬러 | 비고 |
|------|------|------|
| R 자 | 딥 그린 `#2D6A4F` | |
| 하트/심장 | 소프트 코랄 `#E07A5F` | 감성·당사자 강조 |
| 배경 | 크림 `#F5F0E8` 또는 투명 | |

### 강점 / 약점

| 강점 | 약점 |
|------|------|
| 브랜드명 어원과 완전 일치 | 하트 남용으로 진부해 보일 위험 |
| 이니셜 로고로 응용 범위 넓음 | R 단독 사용 시 브랜드 인식 낮음 |
| B2G·임팩트 투자 피칭에 품격 | 복잡한 표현은 소형에서 깨질 수 있음 |
| 워드마크 없이도 브랜드 인식 가능 (장기) | |

## Direction D — 나선형 타임라인 (Spiral Timeline)

### 컨셉

중심에서 바깥으로 펼쳐지는 나선(螺旋) 형태.
나선의 시작점은 영유아기, 끝점은 성인기 자립.
나선이 열려있는(open-ended) 형태로 "계속되는 삶"을 표현.
동시에 기록이 쌓여 두터워지는 층위를 시각화.

### 스케치

```
       ╭─────────╮
      ╭╯  ╭────╮ ╰╮
     ╭╯  ╭╯ ·  ╰╮ ╰╮
     │  ╭╯      ╰╮  │
     │  │  START  │  │
     │  ╰╮      ╭╯  │
     ╰╮  ╰╮    ╭╯  ╭╯
      ╰╮  ╰────╯  ╭╯
       ╰─────────→

  · = 시작점 (영유아기)
  → = 열린 끝 (계속되는 자립)
```

**단순화 버전 (실제 로고용)**
```
     ╭──╮
    ╭╯  │
   ╭╯ · │    Recordare
   │    ╰→   레코다레
   ╰──╮
      ╰→
  (2회전 나선 + 열린 화살표)
```

**앱 아이콘형**
```
  ╭───────╮
  │  ╭─╮  │
  │ ╭╯ │  │
  │ │· ╰→ │
  │ ╰──╮  │
  ╰────╯──╯
```

### 컬러 적용

| 요소 | 컬러 | 비고 |
|------|------|------|
| 나선 시작 (내부) | 소프트 코랄 `#E07A5F` | 영유아기·보호 |
| 나선 중간 | 그라데이션 전환 | 성장 과정 |
| 나선 끝 (외부 화살표) | 딥 그린 `#2D6A4F` | 자립·완성 |

### 강점 / 약점

| 강점 | 약점 |
|------|------|
| "끊기지 않는 기록"을 가장 강력하게 표현 | 복잡도 높아 소형에서 인식 어려움 |
| 생애주기의 흐름과 성장이 동시에 | 디자이너 역량에 따라 품질 편차 큼 |
| 독창성 높음, 유사 로고 가능성 낮음 | 단색 버전에서 형태 단순화 필요 |
| 그라데이션 활용으로 시각적 풍부함 | |

## Direction E — 기억의 실 (Thread of Memory)

### 컨셉

여러 사람(당사자·보호자·교사·지원사)을 하나의 실로 연결하는 형태.
실의 가운데 매듭은 당사자를 중심에 놓는 철학을 표현.
실(Thread)은 동시에 기록의 흐름·연속성을 상징.
따뜻하고 수공예적인 감성으로 커뮤니티 플랫폼 느낌.

### 스케치

```
  ·           ·
   \         /
    \       /
     ·─────·      ← 매듭 (당사자 중심)
    /       \
   /         \
  ·           ·

  각 끝점 = 이해관계자 (보호자/교사/지원사/기관)
  매듭    = 당사자 (모든 기록의 중심)
  실      = 연결된 기록
```

**세련된 버전**
```
   ╮     ╭
    ╲   ╱
     ╲ ╱
      ╳      Recordare
     ╱ ╲     레코다레
    ╱   ╲
   ╯     ╰

  (X 형태의 교차점이 당사자, 4방향 실이 이해관계자)
```

**앱 아이콘형**
```
  ╭───────╮
  │·     ·│
  │ ╲   ╱ │
  │  ╲ ╱  │
  │   ●   │    ← 중심점(당사자) 강조
  │  ╱ ╲  │
  │ ╱   ╲ │
  │·     ·│
  ╰───────╯
```

### 컬러 적용

| 요소 | 컬러 | 비고 |
|------|------|------|
| 실 (연결선) | 딥 그린 `#2D6A4F` | |
| 중심 매듭 (당사자) | 소프트 코랄 `#E07A5F` | 당사자 중심성 강조 |
| 끝점 (이해관계자) | 딥 그린 `#2D6A4F` 작은 원 | |

### 강점 / 약점

| 강점 | 약점 |
|------|------|
| 당사자 중심 철학을 형태로 직접 표현 | 공예·섬유 브랜드로 오해 가능 |
| 커뮤니티·연결·따뜻함 동시 표현 | 권위·신뢰 표현이 약할 수 있음 |
| 색각 이상에도 형태로 완전 인식 | B2G 피칭에서 전문성 부족해 보일 수 있음 |
| 소형에서도 선명한 X 형태 유지 | |

## 디자이너 전달 브리프

### 필수 산출물 (디렉션별)

```
각 방향별:
□ 심볼 단독 (투명 배경, SVG)
□ 가로형 조합 (심볼 + 워드마크, SVG + PNG)
□ 세로형 조합 (심볼 위 + 워드마크 아래, SVG + PNG)
□ 앱 아이콘 (1:1, 1024×1024 PNG)
□ 파비콘 (32×32, 16×16 ICO)
□ 단색 버전 (흑백, SVG)
□ 역전 버전 (딥 그린 배경 + 크림/흰 심볼, SVG)
```

### 금지 사항

```
✗ 지나치게 복잡한 그라데이션 (인쇄 시 품질 저하)
✗ 얇은 선 1px 이하 (소형에서 사라짐)
✗ 텍스트 아웃라인 미처리 (폰트 미설치 환경 오류)
✗ 심볼과 워드마크 비율 임의 변경
✗ 승인되지 않은 컬러 사용
```

### 크기 기준

| 용도 | 최소 크기 | 형태 |
|------|---------|------|
| 앱 아이콘 | 44×44px | 심볼 단독 |
| 파비콘 | 16×16px | 심볼 단독 (극단순화) |
| 명함 | 25mm 폭 이상 | 가로형 조합 |
| 현수막 | 제한 없음 | 가로형 또는 세로형 |
| OG 이미지 | 1200×630px | 가로형 + 슬로건 |

## 최종 선정 권고

```
1순위: Direction A (잎맥 타임라인)
  → 생애주기·성장·따뜻함이 브랜드 철학과 가장 일치
  → B2C/B2B 모두 적용 가능한 범용성
  → 컬러 포인트 유연 적용 가능

2순위: Direction C (R + 심장)
  → 브랜드명 어원과 완전 일치, 장기 브랜드 자산화 유리
  → B2G·임팩트 투자 피칭에 강점

보조 활용: Direction D (나선형)
  → 풀페이지 랜딩 히어로 배경, 캠페인 이미지 등 대형 비주얼에 활용
  → 단독 로고보다는 브랜드 모티프로 사용 권장
```
