## 1. 브랜드 네임

### 1.1 정식 명칭

| 구분 | 표기 |
|------|------|
| 영문 (정식) | **Recordare** |
| 한글 (표기) | **레코다레** |
| 영문 약자 | **RC** |
| 도메인 방향 | `recordare.kr` / `recordare.io` |

### 1.2 어원 및 의미 레이어

```
Recordare
│
├── 라틴어 "recordari" — 기억하다, 회상하다 (to remember, to recall)
│     └── re(다시) + cor(심장·마음) = "마음에 다시 새기다"
│
├── 영어 "Record" — 기록하다, 기록
│     └── 시각적으로 즉시 인식되는 현대적 기능 표현
│
└── 어미 "-are" — 라틴어 동사 원형 어미
      └── 이탈리아어·포르투갈어에서도 "~하다"를 뜻하여 글로벌 친숙감
```

**핵심 의미 요약**: 단순한 데이터 저장이 아닌, **마음에 새기는 살아있는 기억** — 지적장애인 당사자의 삶이 기록을 통해 연결되고 보존되며, 그 기억이 자립의 토대가 됨을 표현.

### 1.3 발음 가이드

| 언어 | 발음 | 표기 |
|------|------|------|
| 한국어 | 레-코-다-레 | 4음절, 균등 강세 |
| 영어 | reh-cor-DAH-reh | 3번째 음절 강세 |
| 라틴어 원음 | reh-KOR-dah-reh | 2번째 음절 강세 |

> **권장**: 한국 서비스이므로 한국어 발음 "레코다레" 균등 4음절 사용.

## 2. 브랜드 철학

### 2.1 브랜드 본질 (Brand Essence)

> **"마음에 새기는 기억, 스스로 자라는 삶"**

지적장애인 당사자의 삶은 기관이 바뀔 때마다 처음부터 다시 소개되어야 했습니다.
Recordare는 그 기억이 끊기지 않도록 마음에 새기고, 그 기억이 당사자 스스로의 권리로 자라도록 지원합니다.

### 2.2 브랜드 미션

> **"모든 지적장애인 당사자의 삶이 기억되고, 그 기억이 자립의 힘이 되도록"**

### 2.3 브랜드 비전

> **"2030년, 국내 모든 발달장애 관련 기관에서 Recordare가 표준 기록 플랫폼이 된다"**

### 2.4 브랜드 가치 (Core Values)

| 가치 | 의미 | 실천 방식 |
|------|------|---------|
| **연속성 (Continuity)** | 기록은 끊기지 않는다 | 기관 전환 시 인계서 자동 생성 |
| **존엄 (Dignity)** | 당사자는 기록의 주인이다 | 성인 전환 시 권한 당사자 이행 |
| **신뢰 (Trust)** | 민감한 정보를 안전하게 | ISMS-P 인증, 암호화, Audit Log |
| **접근성 (Accessibility)** | 누구나 쉽게 | AAC 픽토그램, 음성 안내, 쉬운 말 |
| **성장 (Growth)** | 기록이 쌓여 자립이 된다 | 생애주기별 발달 추이 시각화 |

### 3.1 메인 슬로건 (Main Slogan)

> ## "당신의 삶, 단 하나의 기억으로"

**선정 이유**:
- "당신" — 당사자 중심성 명확히 표현
- "단 하나" — 파편화 해소, 통합의 가치
- "기억으로" — Recordare 어원(마음에 새기다)과 직결
- 보호자·당사자·전문가 모두에게 공명하는 보편적 언어

## 5. 브랜드 퍼스낼리티

### 5.1 브랜드 성격 (Brand Personality)

Recordare는 다음 5가지 성격을 가집니다:

| 성격 | 설명 | 반대 (지양) |
|------|------|-----------|
| **따뜻함 (Warm)** | 차갑고 기술적인 느낌이 아닌, 사람 중심의 온기 | 냉철, 기계적 |
| **신뢰감 (Trustworthy)** | 민감한 정보를 다루는 만큼 안정적이고 믿음직함 | 가볍고 실험적 |
| **전문성 (Expert)** | 법적·복지 도메인 전문성을 갖춘 파트너 | 아마추어적 |
| **존중 (Respectful)** | 당사자를 보호 대상이 아닌 권리의 주체로 대함 | 시혜적, 동정적 |
| **명료함 (Clear)** | 복잡한 법·제도를 쉽게 안내 | 모호하고 어려운 |

### 5.2 브랜드 목소리 (Brand Voice)

| 상황 | 톤 | 예시 |
|------|-----|------|
| 랜딩 페이지 | 공감 + 희망 | "더 이상 처음부터 다시 시작하지 않아도 됩니다." |
| 온보딩 | 따뜻 + 안내 | "걱정 마세요. 차근차근 함께 할게요." |
| 오류 메시지 | 친근 + 명확 | "잠깐, 연결이 끊겼어요. 걱정 마세요, 기록은 안전합니다." |
| 권한 경고 | 진중 + 명확 | "OO 선생님의 접근 권한이 7일 후 만료됩니다." |
| 당사자 UI | 쉬운 말 + 밝음 | "잘 했어요! 선생님이 이제 알게 됐어요." |
| B2B 영업 | 전문 + 신뢰 | "ISMS-P 인증 기반, 법적 요건을 충족하는 유일한 플랫폼." |

### 5.3 금지 표현 (Voice Off-Limits)

- "장애우" → "당사자" 또는 "지적장애인"으로 표현
- "불쌍한", "힘든" 등 동정적 표현 지양
- "관리" → "기록·지원"으로 표현 (당사자를 관리 대상화하지 않음)
- "정상인" 등 비교·차별적 표현 전면 금지
- 지나치게 기술적인 약어 남발 (STT, LLM 등을 일반 사용자에게 직접 노출 지양)

## 6. 시각적 아이덴티티 방향

### 6.1 로고 컨셉 방향

**방향 A — 잎(Leaf) 타임라인**
```
  ╭─╮
 ╭╯ ╰─╮        Recordare
╰╯    ╰╯       레코다레
  ──────
생애주기를 상징하는 잎맥 구조.
각 줄기가 영유아→학령기→전환기→성인기를 표현.
```

**방향 B — 연결 고리 (Link)**
```
  ○──○──○──○   Recordare
  ↑  ↑  ↑  ↑   레코다레
  기관간 기록 연결을 시각화한 심플 아이콘.
  원의 크기가 성장함을 표현.
```

**방향 C — R 이니셜 + 하트**
```
  ℝ             Recordare
                레코다레
라틴어 어원(마음에 새기다)에서 착안,
R 자 안에 심장(cor) 형태를 내포.
```

> **권장**: 방향 A (잎 타임라인) — 생애주기 시각화, 성장 은유, 자연적 따뜻함이 브랜드 가치와 가장 일치.

### 6.2 컬러 팔레트 방향

| 역할 | 컬러 방향 | 의미 |
|------|----------|------|
| **Primary** | 딥 그린 계열 `#2D6A4F` 방향 | 성장·자연·신뢰·지속성 |
| **Secondary** | 웜 크림 `#F5F0E8` 방향 | 따뜻함·안정·기록지 연상 |
| **Accent** | 소프트 코랄 `#E07A5F` 방향 | 희망·당사자 중심·생동감 |
| **Alert** | 앰버 `#F2A93B` 방향 | 권한 만료 알림·주의 |
| **Text** | 차콜 `#2D2D2D` | 가독성·진중함 |

> **색각 이상 대응**: Primary-Secondary 조합은 색맹 시뮬레이션 통과 필수 (WCAG 2.1 AA 대비율 4.5:1 이상).

### 6.3 타이포그래피 방향

| 용도 | 한글 서체 방향 | 영문 서체 방향 |
|------|------------|------------|
| 헤드라인 | 프리텐다드 Bold / 나눔스퀘어 ExtraBold | Cormorant Garamond (품격·라틴 어원 반영) |
| 본문 | 프리텐다드 Regular | Inter Regular |
| AAC 모드 | 나눔고딕 ExtraBold (큰글씨) | — |
| 법적 고지 | 프리텐다드 Light | Inter Light |

### 6.4 이미지·일러스트 방향

- **따뜻한 일러스트 스타일** (사진보다 일러스트 우선) — 실제 장애인 사진 사용 시 당사자 동의 및 존엄 훼손 방지
- **라인아트 + 파스텔 채색** — 기록지·노트 감성 연계
- **생애주기 타임라인 시각화** — 씨앗→새싹→꽃→나무 성장 은유
- **픽토그램 우선** — AAC 기반 UI 일관성 유지
