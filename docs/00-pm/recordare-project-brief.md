## §1. 무엇을 만드나? (What)

Recordare는 **지적장애인 당사자의 삶이 기관·시기·전문가가 바뀌어도 끊기지 않도록 기록을 잇는 평생 기록 플랫폼**입니다.

- 영유아기(0~6세)부터 고령기(65세+)까지 **5단계 생애주기**가 하나의 타임라인으로 이어집니다.
- 보호자·당사자·활동지원사·특수교사·일반교사·사회복지사·시설장·후견인 **8종 이해관계자**가 한 플랫폼에서 협업합니다.
- 입력은 **빠른 선택(15초)·체크리스트(5초)·사진+태그(30초)·텍스트** 4종 모드로 환경에 맞춰 선택 가능합니다.
- 권한은 **5×20→6×20 = 120셀 매트릭스**로 미세 조정되고, 18세가 되면 6개월간 단계적으로 당사자에게 이양됩니다.
- 자가진단(K-DST·M-CHAT-R·AIR-SDS) 결과는 시계열로 누적되어 AI가 추이 인사이트를 제공합니다.

> 한 줄로: **"기록은 끊기지 않고, 권한은 당사자에게 자란다."**

## §2. 왜 지금인가? (Why Now)

### 2.1 시장의 절실함

| 지표 | 값 | 출처 |
|------|-----|------|
| 국내 등록 발달장애인 | **26.2만 명** | 보건복지부 2024 |
| 가족 포함 직접 영향 인구 | **78만 명** | 보건복지부 2024 |
| 그룹홈·거주시설 수 | **8,000곳** | 사회복지시설 정보시스템 |
| 활동지원사 자격 보유자 | **4만 명** | 한국활동지원사협회 |
| 발달재활서비스 바우처 (연 예산) | **1,243억 원** | 보건복지부 2024 |
| 권한 이양 절차 부재 가정의 시설 재진입율 | **+35%** | 보건복지부 자립지원 통계 |
| 신규 진단 보호자의 K-DST 미사용 비율 | **38%** | 보건복지부 2024 |

### 2.2 정책 모멘텀

- **발달장애인 권리보장 및 지원에 관한 법률** — 의사 존중·생애주기 지원 명문화
- **장애인 등에 대한 특수교육법 §22** — 개별화교육(IEP) 의무
- **개인정보보호법 §35의2** — 데이터 이동권 (2024 개정)
- **민법 §938~959** — 한정/특정/성년 후견 3종 차등
- **사회서비스원** — 바우처 청구 디지털화 가속 (2025~)

### 2.3 기술 임계점 도달

- LLM·STT·OCR·AI Vision의 한국어 정밀도가 시범 운영 가능 수준
- PWA·IndexedDB·Service Worker 성숙으로 산간·시설 오프라인 환경 대응 가능
- 본인인증·전자세금계산서·대법원 가족관계등록 검증 API 공개

> **"기술이 준비됐고, 정책이 열렸고, 사용자 데이터가 누적되고 있다. 이제는 통합할 시점이다."**

## §4. 어떻게 다른가? (How)

### 4.1 단독 사분면 — 권한·연속성 × 생애주기·자기결정

```
        권한·연속성 高
              ↑
              │
              │       ★ Recordare (단독)
              │       (6×20 권한 + 5단계 생애주기 + AAC)
              │
   ───────────┼───────────→ 권한·연속성 低
              │
   행복e음 ●  │  ● 케어아이
   NEIS ●     │
              │  ● 누리 / 에이블링 / 베어베터 / 우리집 케어
              ↓
```

### 4.2 경쟁 분석 (민간 5 + 공공 5)

| 구분 | 서비스 | 차별점 |
|------|--------|--------|
| 민간 | 케어아이 (그룹홈 ERP) | 인계서·당사자 모드 부재, 폐쇄 DB |
| 민간 | 누리 자립지원 | 일지만, 권한 매트릭스 없음 |
| 민간 | 에이블링 | 커뮤니티 중심, 기록 약함 |
| 민간 | 베어베터 라이프 | 직장 일지 한정 |
| 민간 | 우리집 케어 노트 | 개인 메모, 권한·인계 부재 |
| 공공 | 행복e음 (보건복지부) | v1.1 이연 연동 — 사회복지 케이스 |
| 공공 | NEIS (교육부) | v1.1 이연 연동 — IEP·학적 |
| 공공 | 발달재활서비스 정보 | FR-56 B2G 청구 자동화 |
| 공공 | 국민건강보험 의료정보 | v2.0 이연 |
| 공공 | 장애인 등록 정보 | 본인인증 시 자동 매칭 |

### 4.3 3축 차별화

1. **권한 매트릭스 6×20 = 120셀** — 단일 최대 정밀도, 미세 조정
2. **생애주기 5단계** — 영유아·학령·전환·성인·고령 평생 연속성
3. **AAC·쉬운말 당사자 모드** — 1화면 1결정·2단계 확인·시간 압박 0

### 6.3 단위 경제성 (예상)

- **B2C ARPU**: 자녀당 9,900원/월 × 12개월 = **연 118,800원**
- **B2B ARPU**: 시설당 약 10명 × 11,000원 × 12개월 = **연 132만원/시설**
- **CAC 가설**: B2C 인스타+협회 추천 ≤ 15,000원 / B2B 시설장 직접 영업 ≤ 50만원
- **Payback**: B2C 약 1.5개월 / B2B 약 4개월
- **LTV (가설)**: 가정 평균 5년 잔존 = 약 60만원 / 시설 평균 7년 = 약 925만원

## §12. 시설장·보호자를 위한 안심 설명

> 본 절은 시범 시설장·보호자가 도입 결정에 필요한 핵심 정보를 정리합니다.

### 12.1 보호자에게 — "안심하고 시작하세요"

**Q1. 우리 아이 사진 1만 장이 외부로 새지 않을까요?**
- 모든 데이터는 한국 NCP (Naver Cloud) 서버에 AES-256 암호화로 저장됩니다.
- 개인정보보호법 §23 민감정보 별도 동의 + §35의2 데이터 이동권 모두 적용됩니다.
- 누가·언제·어떤 기록을 봤는지 항상 알림으로 받습니다 (FR-14).
- 보호자가 원하면 30일 안에 모든 데이터를 완전 삭제할 수 있습니다 (NFR-07).

**Q2. 자가진단 결과가 학교/시설에 의료 진단처럼 오해되지 않을까요?**
- 모든 자가진단 화면·PDF에 "보호자 관찰 의견 / 정식 진단은 전문의 상담" 면책이 자동 표기됩니다 (FR-68).
- AI 인사이트에는 "진단·치료·의심" 같은 의료 표현이 금지됩니다 (LLM 가드레일).
- 외부 공유 시 PDF에 워터마크가 자동 삽입됩니다.

**Q3. 아이가 18세가 되면 어떻게 되나요?**
- D-180부터 6개월간 단계적으로 자녀에게 권한이 이양됩니다.
- D-150에 모의 동의 시뮬레이션, D-120에 변호사 무료 상담을 안내합니다.
- D-0에 자동 전환되며, 응급 상황을 위해 보호자 의료 알림 수신권은 30일 잔존합니다.
- 자녀가 의사 표명이 어렵다면 후견 모드로 안전하게 전환됩니다.

**Q4. 다른 부모님들과 연결되고 싶어요.**
- 인증된 보호자 전용 자조모임이 있습니다 (v2.5).
- 생애주기 자동 매칭 그룹 + 자유 개설 그룹 모두 가능합니다.
- 자녀 기록·권한·인계서 데이터는 자조모임 UI에 절대 노출되지 않습니다 (FR-83).

### 12.2 시설장에게 — "B2B 도입 5가지 안심 포인트"

| # | 포인트 | 근거 |
|---|--------|------|
| 1 | **ISMS-P 인증 + KISA 위촉 침투 테스트 연 2회** | NFR-03 |
| 2 | **시설장 검토 모달 의무화** — B2G 청구 자동 발행 금지 | Risk #8 대응 |
| 3 | **권한 일괄 처리 시 "N명 영향" 미리보기 + 30일 일괄 되돌리기** | Risk #6 대응 |
| 4 | **인계서 일괄 승인 + B2G 바우처 자동 식별** | FR-23 |
| 5 | **감사 로그 1-Click CSV 내보내기** — 분기 컴플라이언스 보고 | NFR-07 |

### 12.3 활동지원사·일반교사에게 — "환경 무관 15초 입력"

- 시설 점심 식당·교실·이용자 옆에서 STT 없이 빠른 선택 15초로 일지 작성 가능.
- 산간 캠프 등 오프라인에서도 로컬 큐 저장 후 자동 동기화.
- 통합학급 일반교사는 1줄 메모 30초로 특수교사·보호자와 공유.
