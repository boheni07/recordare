## 2. 강점 (Strengths)

### 2.1 PRD 누적 구조의 완성도 — ⭐⭐⭐⭐⭐
- v1.0 → v2.0 → v2.1 → v2.2까지 **결정 컨텍스트를 모두 보존**한 누적 PRD. Changelog C1~C18로 모든 변경 추적 가능
- 68개 FR + 10개 NFR로 **24개월 로드맵 전체 정의 완료**
- §15 18세 이양·§18 후견 모드 등 **법령 기반 정밀 설계** (한국 민법 §938~§959 직접 인용)

### 2.2 페르소나 다양성 — ⭐⭐⭐⭐⭐
- 5종 핵심 페르소나 + 2종 보조 페르소나 (시설장·후견인) = **7종 페르소나**
- v2.2에서 "소속 없음 (가정 거주)" 케이스 (김하준) 추가로 **신규 진단 보호자 First Value 강화**
- 각 페르소나의 JTBD 6-Part 정의 + Pain Point + 인용구 명시

### 2.3 비치헤드 & GTM 명확성 — ⭐⭐⭐⭐
- 비치헤드: **수도권 그룹홈 30곳 + 신규 진단 보호자 100명** — 명확한 진입 세그먼트
- 3단계 GTM (M1-M6 비치헤드 → M7-M18 확장 → M19-M36 표준화) 단계별 KPI 명시
- TAM 4,500억 / SAM 1,200억 / SOM 180억 — 출처 (보건복지부 2024) 명시

### 2.4 자가진단 v2.2 추가의 전략적 가치 — ⭐⭐⭐⭐⭐
- 신규 진단 보호자 페르소나의 페인을 **객관적 도구로 해소** — First Value 강화
- §15 18세 이양과 자기결정능력 자가체크 (AIR-SDS) 직접 연결 — 데이터 흐름 정합
- 의료법 안전 면책 (FR-68) 사전 명시 — Risk #10 자체 차단 설계

## 3. 약점 / 개선점 (Weaknesses)

### 3.1 🔴 Critical — 시장 규모 추정의 근거 약함
- TAM/SAM/SOM 산정 근거가 "보건복지부 2024 통계" 만 인용. **세부 계산 가정 미공개**
  - 예: SOM 180억 = 시설 300 × 평균 60만원 × 12개월 인가? 명시 필요
- **권장**: PRD v2.3에 §3.5 시장 산정 워크시트 추가 (가정·환산식·민감도 분석)

### 3.2 🔴 Critical — 경쟁사 분석 깊이 부족
- 5종 카테고리 비교는 명료하나, **국내 직접 경쟁 SaaS** 분석 0건
  - "케어아이" 외 구체 사례 누락. KASKO, 행복e음 등 공공 시스템과의 **데이터 호환성** 검토 부재
- **권장**: 보건복지부 산하 발달장애 정보시스템 5종 + 민간 케어 SaaS 3종 추가 분석 (v2.3)

### 3.3 🟡 Important — 가격 민감도 검증 미흡
- Personal Plus 9,900원 / Facility 9,500~12,000원 — **숫자만 있고 PoC 결과 없음**
- PoC E3 "시설 가격 민감도" (6주)가 마일스톤에만 있고 실제 진행 계획 부재
- **권장**: M3 이전 시설 5곳 사전 인터뷰 (가격 수용성 + WTP 측정)

### 3.4 🟡 Important — 핵심 가설 H1~H5 측정 방법 추상적
- 가설별 통과 기준은 명시되나, **측정 도구·기간·표본 크기** 미정
  - 예: H5 "당사자 자문단 평가 4/5+" — 5명이 충분한 표본인가? (통계적 유의성 부족)
- **권장**: 가설별 측정 프로토콜 명세서 별도 작성 (실험 디자인 + 통계 분석 계획)

### 3.5 🟡 Important — 정부 정책 변화 리스크 명시 부재
- 발달장애인 권리보장법 개정·발달재활서비스 바우처 정책 변화가 **B2G 가설 전체 흔들 수 있음**
- 현재 PRD에는 정책 의존성 및 시나리오 분석 0건
- **권장**: Pre-mortem Risk #12 추가 — "정부 정책 변화로 B2G 청구 모델 무효화"

### 3.6 🟡 Important — 글로벌 확장 가능성 미평가
- "한국어 우선, 영어 v2" 외 글로벌 전략 부재
- 일본 (성년후견제도 동일) / 대만 (유사 발달장애 정책) **인접 시장 진입 가능성** 미분석
- **권장**: v3.0 글로벌 로드맵 별첨 (선택)
